| Command | Description |
| --- |  ---  |
| Option+ F7 | 搜索对象被使用的地方 |
| Ctrl+Option+H | 查看方法引用关系 |
| Cmd+Shift+H | 显示方法的层级结构 |
| Cmd+O | 打开类 |
| Control+O | 重写方法 |
| Cmd+Shift+Enter | 自动补全末尾括号、分号等 |
| Cmd + Plus,Minus | 折叠展开代码块 |
| Cmd + Shift + Plus,Minus | 折叠展开全部代码块 |
| F2 | 快速定位错误位置 |
| Cmd+B | 跳到方法声明 |
| Cmd+Option+B | 跳到方法实现 |
| Cmd+E | 最近打开文件 |
| Option+上/下 | 区块智能选择 |
| Cmd+Option+L |  格式化代码 |
| Cmd+P | 方法参数提示 |
